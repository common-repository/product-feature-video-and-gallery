This plugin enables to add feature video to product.

== Description ==

VideoShop - Product Feature Video and Gallery - Add feature video to product shop page, category page, archive page and add feature video along with images on product single page. You can add feature video to any product and can add multiple video to make product video gallery at the product single page. You can also mix video and images combine as product gallery. With multiple Product Video you can Instantly transform your product single page into a fully Responsive video Carousel. This plugin will work with any WordPress theme. Shortcode is included for Elementor and Divi Builder. It is also supported variable product, so that if you add different image to each variation and on selecting that variation right image come up on viewport.

**Plugin Features:**

*   Add YouTube video as feature video to any product
*   You Can Add feature video along with feature image and gallery image
*   Support product variation image changing
*   Videos will show in the gallery if the product doesn't have any image.
*   Adaptive video height based on single product image size
*   RTL support

== Frequently Asked Questions ==

= Is free version support local self hosted video? =

No. Pro version support Local self hosted video, Vimeo video and any video with embeded link.

= Is free version support shop page feature video? =

No. Free version only support product single page feature video.
